# Proyecto Tecno Quality

**Tecno Quality** es una aplicación web diseñada para la gestión integral de conductores, personal administrativo y documentación técnica relacionada con evaluaciones médicas, psicotécnicas, psicologicas y credenciales.

## Descripción

El sistema permite:

- **Gestión de usuarios:** Registro, edición y eliminación de usuarios según roles (Administrador, Secretaría, Psicotécnico, etc.).
- **Gestión de conductores:** Registro, consulta y edición de datos de conductores.
- **Documentación técnica:** Subida, descarga y gestión de documentos médicos, psicotécnicos y credenciales.
- **Control de acceso:** Autenticación y autorización basada en roles.

## Tecnologías

- **Frontend:** React.js, React Router, Tailwind CSS
- **Backend:** Java Springboot
- **Gestión de estado:** Context API, Hooks de React
- **Autenticación:** Token JWT, persistencia local


## Uso

1. **Iniciar sesión:** Accede a `/login` para autenticarte.
2. **Navegar según tu rol:** El sistema te redirigirá a la vista correspondiente según tu rol.
3. **Gestión de usuarios y conductores:** Utiliza los botones y formularios para registrar, editar o eliminar usuarios y conductores.
4. **Subir y descargar documentos:** Accede a las secciones de documentación para gestionar archivos médicos, psicotécnicos y credenciales.
